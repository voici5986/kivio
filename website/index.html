能力

十二项能力，一个工作台之内

以智能体为核心，从对话、工具到检索层层展开；屏幕级的轻功能依旧安静地守在热键之后。

01+

智能体循环

规划、执行、综合。文件、Shell、网页、后台任务一气呵成。

02+

并行子智能体

一条消息派发多个专职人格，并发执行，结果内联返回。

03+

MCP 工具

接入任意 Model Context Protocol 服务器，工具即插即用。

04+

Skills

Markdown 定义技能，运行中激活，可挂载可执行脚本。

05+

知识库 RAG

向量 + 全文混合检索，RRF 融合，可选重排。全部本地索引。

06+

代码沙箱

Pyodide 本地跑 Python：pandas、matplotlib、文档生成。

07+

记忆与规划

跨会话记忆、任务清单与计划追踪，长任务不再断片。

08+

任意服务商

四种原生协议直连；每家可配密钥池，限额自动轮换。

09+

连接器

Notion 等远程数据源一键 OAuth 直连，本地保管令牌。

10+

划词翻译

全局热键，流式出译文，提交即回填原应用。空闲仅 ~50MB。

11+

Lens 取景问答

圈选屏幕任意区域或窗口，画面直接进入视觉对话。

12+

三引擎 OCR

Apple Vision、Windows OCR、离线 RapidOCR，任何环境可用。
